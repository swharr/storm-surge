This middleware integrates feature flag providers (LaunchDarkly, Statsig) with Spot Ocean for dynamic infrastructure management.

Available Endpoints:

  • GET /health - Health check endpoint
  • POST /webhook - Feature flag webhook handler
  • GET /api/flags - List feature flags
  • GET /api/clusters - List Spot Ocean clusters
  • GET /api/costs/metrics - Cost metrics
  • POST /api/auth/login - User authentication

For API documentation, visit: /api/health

Dashboard access requires authentication via the API.
